# SOLUMITOS — Deploy a Netlify ## Estructura del repo ``` solum-crm/ ├── index.html ← El CRM completo ├── netlify.toml ← Configuración de Netlify ├── netlify/ │ └── functions/ │ └── claude.js ← Proxy para la API de Claude └── README.md ``` ## Pasos para subir a GitHub 1. Abrí el repo: github.com/mmarinelliprop-lgtm/solum-crm 2. Subí estos 4 archivos (arrastrando o con el botón Upload): - `index.html` - `netlify.toml` - `netlify/functions/claude.js` - `README.md` ## Agregar la API Key en Netlify 1. Entrá a netlify.app → tu sitio solumitos 2. Site configuration → Environment variables 3. Agregar variable: - Key: `ANTHROPIC_API_KEY` - Value: tu API key (sk-ant-...) 4. Guardar → Netlify redespliega solo ## Listo Abrí solumitos.netlify.app desde cualquier dispositivo. El chat ya funciona — la API key está guardada en Netlify, no en el código. ## Para actualizar el HTML en el futuro Solo subís el `index.html` nuevo al repo → Netlify redespliega automáticamente. La API key no hay que tocarla nunca más.
